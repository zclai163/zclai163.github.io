七 git stash

  • 显示保存进度的列表

  • git stash pop [–index] [stash_id]

  • git stash apply [–index] [stash_id]（除了不删除恢复的进度之外，其余和git stash pop 命令一样。）

  • git stash drop [stash_id]（删除一个存储的进度。如果不指定stash_id，则默认删除最新的存储进度。）

  • git stash clear（删除所有存储的进度。）
